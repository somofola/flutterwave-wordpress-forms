# Payment Forms for Flutterwave

WordPress plugin to create payment forms billed via Flutterwave. Supports one-time payments, recurring subscriptions, quantified purchases, donations, and CSV exports.

**Developer:** Shola Omofola
**Repository:** https://github.com/somofola/flutterwave-wordpress-forms

## Installation

1. Download the plugin ZIP from the [GitHub repository](https://github.com/somofola/flutterwave-wordpress-forms).
2. In your WordPress dashboard, go to **Plugins → Add New → Upload Plugin**, choose the ZIP, and click **Install Now**.
3. Activate the plugin. A database table `{wp_prefix}flutterwave_forms_payments` is created automatically.
4. Configure your Flutterwave public/secret keys under **Settings → Flutterwave Forms**.
5. Create a form, copy the shortcode, and paste it into any page or post.
